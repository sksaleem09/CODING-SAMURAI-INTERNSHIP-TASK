# ATM Banking System - Background Images Guide

## Required Images

Your ATM Banking System requires the following background images (all should be 800x600 pixels for best results):

1. **welcome.jpg** - Welcome/Home screen background
2. **create_account.jpg** - Account creation screen
3. **pin.jpg** - PIN generation screen
4. **login.jpg** - Login screen
5. **menu.jpg** - Main menu screen
6. **balance.jpg** - Balance enquiry screen
7. **deposit.jpg** - Deposit screen
8. **withdraw.jpg** - Withdraw screen

## Image Specifications

- **Recommended Size**: 800x600 pixels (matches the JFrame size)
- **Format**: JPG or PNG
- **Style**: Professional, modern, banking/ATM themed
- **Color Scheme**: Dark blues, grays, or professional gradients work best
- **Text**: NO text should be inside the images (all text is added via Java labels)

## Tips for Professional Look

1. **Consistent Theme**: Use similar color schemes across all screens
2. **Dark Backgrounds**: Dark backgrounds make white text more readable
3. **Subtle Patterns**: Avoid busy patterns that distract from UI elements
4. **Modern Feel**: Use contemporary, clean designs
5. **Banking Theme**: Images should relate to banking/finance/ATM concepts

## Quick Start

If you want to test without images first:
- The code will automatically use gradient backgrounds if images are missing
- You can add images later without changing any code
- Just place the images in the `images/` folder with the exact names listed above
